Sobre mim

Meu nome é Alexsandro, sou bolsista do curso FullStack Web Developer da Growdev em parceria com a Sicredi Pioneira e este é o meu portfolio de práticas e projetos desenvolvidos durante o curso.

Estou animado para compartilhar meu trabalho e estou aberto a feedback e oportunidades de colaboração. Sinta-se à vontade para explorar meu portfólio e entrar em contato comigo para discutir como podemos trabalhar juntos para alcançar nossos objetivos comuns no mundo do desenvolvimento web.

Sobre o curso

O Programa de Starter Full Stack Web Developer é uma formação completa que vai dos fundamentos da programação e tecnologia até o desenvolvimento de um software completo com front-end, back-end, banco de dados, com qualidade garantida com testes automatizados e a disponibilização (deploy) do sistema em produção. Nesta edição, iremos aprender tecnologias utilizadas na Sicredi Pioneira, como um dos frameworks mais utilizados para desenvolvimento web chamado Laravel e também o Vue.js.
